## Getting Started

Clone the repo and open it locally:

```bash
git clone https://github.com/George-alt5/richellewkihoro.net.git
cd richellewkihoro.net/portfolio
```

Then simply open `index.html` in your browser, or serve it locally, e.g.:

```bash
npx serve .
```
